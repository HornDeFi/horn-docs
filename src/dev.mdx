# Basics

Horn works through smart-contracts on the Ethereum chain with a master contract and a new contract is created for each new pool.

### Master contract 
This is the main Horn contract which refer to every other contracts (pool) of the Protocol. <br/>
You can request it to fetch a list of each pools existing on Horn.

```
0x509c136121E1C60D3EAc26ea20FEEaE1f51959BA
```

You can use web3.js to call manually the existing functions of our contracts
<a href="https://web3js.readthedocs.io/en/v1.3.0/"> Web3.js Docs </a><br/>
